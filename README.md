"# gameBoySoftware" 


This repo is a collection of all of the code that I have been testing out for my Game Boy replica project.

Essentially I have a menu screen / home screen where all of the games are run from.
The games are each stored as separate gamefiles.
I have about 13 games and I created one 3d animation with a cube (multiple versions of the animation too).

I need to finish adding sounds to the game, finish the PCB design, and then 3d print a case and then this project will be more than a mess of wires on my desk!
